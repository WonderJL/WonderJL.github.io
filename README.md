# wonderjl.github.io

Public GitHub Pages mirror of the built portfolio site for Jeff Leung.

**Source repository:** the private `WonderJL/portfolio` repo authors the site.
**This repo:** contains only the built `dist/` artifact, pushed automatically by
the source repo's GitHub Actions workflow on every push to `main`.

Do not edit this repo directly — any commits made here will be overwritten on
the next deploy.

Mirror URL:    <https://wonderjl.github.io>
Canonical URL: <https://jeff-leung-portfolio.pages.dev> (Cloudflare Pages — the host that enforces full CSP and security headers)
